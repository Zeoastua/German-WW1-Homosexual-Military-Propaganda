the juniper tree

German Version
English Version

While beginning to mark-up The Juniper Tree from the original source for the German text, the team discovered that it was in Dutch instead of German. Efforts to find a usable German version that we felt confident in from other sources proved unsuccessful. We have decided to include only the English version of The Juniper Tree as it still offers interesting data as to our main research question of the Grimms' portrayal of women characters.
